# java-poo-loja-celular
Este projeto consiste em um sistema simples de gerenciamento de uma loja de celulares, desenvolvido utilizando Programação Orientada a Objetos (POO) em Java.

A aplicação simula operações básicas de cadastro e manipulação de objetos, com foco na organização de classes, encapsulamento e separação de responsabilidades.

O objetivo principal foi praticar conceitos fundamentais de orientação a objetos e estruturação de código.

- Tecnologias Utilizadas:

> Java
> Programação Orientada a Objetos (POO)
> Estrutura de classes e métodos
> Execução via terminal

- Conceitos Aplicados:
  
> Criação de classes e objetos
> Encapsulamento
> Métodos e atributos
> Organização da lógica de negócio
> Separação de responsabilidades entre classes

Como Executar ->

1. Clone o repositório: git clone https://github.com/kkfabio/java-poo-loja-celular.git
2. Acesse a pasta do projeto: cd java-poo-loja-celular
3. Compile os arquivos: javac *.java
4. Execute o programa: java Main
